#Taucho

Taucho is a GTK+ theme.
If you find a bug or want an improvement in any way, simply raise an issue and we'll be there.

## Support
- GTK 3.20
- GTK 3.18

## Installation

```
$ mkdir ~/.themes
$ git clone https://github.com/dotovr/Taucho.git ~/.themes/Taucho
```
